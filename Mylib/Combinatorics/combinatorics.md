## Operations

- $\mathtt{init(int\ N)}$
	- $0!$ ~ $N!$とその逆数を計算する。
	- Time complexity $O(N)$
- $\mathtt{f(int\ i)}$ : return $i!$
- $\mathtt{finv(int\ i)}$ : return $\frac{1}{i!}$
- $\mathtt{P(int\ n,\ int\ k)}$ : return $_nP_k$
- $\mathtt{C(int\ n,\ int\ k)}$ : return $_nC_k$
- $\mathtt{H(int\ n,\ int\ k)}$ : return $_nH_k = _{n+k-1}C_k$
